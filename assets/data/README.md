# Daily log (published)

`daily-log.json` is the **public** source of truth for [`daily-log.html`](../../daily-log.html) at the site root. Everyone who visits your site sees whatever is in this file after you push to GitHub.

## How to update

1. Open **`daily-log-edit.html`** locally or on your site (bookmark it; it’s not linked from the portfolio).
2. Edit days, then **Download `daily-log.json`**.
3. Replace `assets/data/daily-log.json` in this repo and commit + push.

Optional: edit the JSON by hand in your editor instead of using the edit page.

Private notes (SQL / Python / LeetCode) on the edit page stay in **your browser only** and are **not** saved to this file.
